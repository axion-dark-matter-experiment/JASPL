J.A.S.P.L. (Just Another Signal Processing Library)

Introduction

JASPL is designed to perform signal processing operations on 1 dimensional objects, with an emphasis on GPGPU acceleration.

Dependencies

  • Boost libraries >= 1.62
  • gnuplot-iostream ( included with source files )
  • fftw >= 3.3.5 http://www.fftw.org/download.html

Dependencies

  • OpenCL installation (e.g. Nvidia CUDA, AMD-APP, Intel)
  • clFFT https://github.com/clMathLibraries/clFFT.git

Considerations if compling from source

  • OpenMP Installation and compilier that supports OpenMP pragmas ( GCC )
  • C++11 Compliant Compilier ( GCC >= 4.8.2 )
